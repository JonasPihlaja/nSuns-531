# nSuns App

## Grund funktionalitet: 
Sätt in värden (TM)
Räkna ut motsvarande träningsvikt
Logga +1 set
Gå igenom historia
Uppdatera nästa veckas vikter automatiskt från +1 set
	

## UI:
Instagram "bottom navbar"

"Clickable boxes" per set, visa alla på en gång.
En övning i gången


## Teknologi:	
Bootstrap 5

ES6 JavaScript -> Axios

Node.js -> Express -> .env

(MySQL)


## Deployment:
Nginx
SSL certificates
